Live Data Visualization

Reading live data on Finland's energy production and consumption from Fingrid's open data database and visualization it using D3.js JavaScript library. The data is updated every 3 minutes.

  1. 1.Chose the Start/End time.
  2. 2.chose one of the choices ( Electricity Production / consumption / wind production )
  3. 3.Press GET DATA button
Project code on github

Get Data

Fingrid Oyj is the enterprise which takes care of the functioning of the nation-wide high-voltage grid, the backbone of electricity transmission in Finland. Fingrid produces large amount of data on the electricity market and the power system and publishes it as open data for public use.
